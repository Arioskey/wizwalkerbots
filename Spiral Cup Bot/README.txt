This Spiral Cup bot farms the spiral cup gauntlet for energy gear
requirements: wizwalker, wizsprinter

Uses WizSprinter (credit SirOlaf)

set up configs according to your character's spells e.g. temp[epic] | lord[epic]
for more info about config set up see Major's examples on his github

To start have a character afk spinning in the house that has the gauntlet
Place clients that are farming accordingly from top left corner of screen

edit spiralcupbot.py to place your friends name in <name> (in the function teleport_to_friend args)
also edit spiralcupbot.py to the amount of clients that will be farming gauntlet (excluding afk spinning player)


(if a client is afk spinning put it in the bottom right)

run bot
